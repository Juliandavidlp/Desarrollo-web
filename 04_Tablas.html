Tablas

  • Las tablas se usan para representar datos.
  • El ejemplo de aplicación más común son los documentos de Excel
  • En html tenemos que definir los elementos de cada tabla por medio de etiquetas
  • Las tablas no se usan hoy en día para maquetar porque se maqueta con CSS.

Etiquetas

< table>: Representa todo el elemento de la tabla.

< tr>: Table row/register representa una fila o un registro.

< td>: Table data cell sirve para representar e insertar celdas.

< th>: Table header sirve para reperesentar un encabezado en una celda, para insertar encabezados.

< tbody>

: Table body es para representar el cuerpo de la tabla.

Por ejemplo:

Día Mes
10Agosto
14Septiembre


Los atributos Colspan y Rowspan

Son atributos que permiten que una celda ocupe más de una celda o una fila.
Es lo que comúnmente llamamos “combinar celdas” a la hora de agrupar celdas en una fila o generar una columna.

Colspan: Indica el número de celdas o columnas a abarcar en una fila.

Rowspan: Indica el número de celdas o filas a abarcar en una columna.

Por ejemplo:

Zona Sur Avellaneda Lanús
Banfield
Quilmes Berazategui
