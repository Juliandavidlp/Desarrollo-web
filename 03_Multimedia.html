Imágenes

Para mostrar una imagen en una página tenemos dos formas de hacerlo. Una es usando el elemento < img/ >, otra mediante CSS, que veremos más adelante.
La etiqueta img requiere de dos atributos obligatorios: src (de source) y alt (de alternative).

Usamos alt para:

  • Posicionamiento en buscadores (extrayendo palabras clave)
  • Personas con dificultades visuales (lectores de páginas Web)
  • Chatbots
  • Establecer una descripción sobre la imagen si la misma no se encuentra disponible

Favicon

Un favicon es la pequeña imagen que se muestra en la pestaña del navegador o en la lista de marcadores (favoritos). El tamaño en la barra de direcciones es de 16x16 píxeles.
Para agregarlo debemos tener la imagen .png que deseamos colocar como ícono en formato .ico, que se puede convertir desde https://convertico.com/, y luego agregarla en el head de nuestro documento HTML por medio de la etiqueta link poniendo en el atributo rel la relación icon:

< link rel="icon" href="nombredelarchivo.ico" >


Audio

    La etiqueta < audio > acepta como atributos:

  • preload: es usado en el elemento audio para almacenar temporalmente (buffering) archivos de gran tamaño. + info.
  • src: puede ser la URL de un archivo de audio o una ruta local al mismo. + info
  • controls: muestra los controles estándar para el audio en una página web. + info
  • autoplay: hace que el audio se reproduzca automáticamente. + info
  • loop: hace que el audio se repita en bucle automáticamente. + info
  • muted: especifica que la salida de audio debe estar silenciada. + info

Video

    La etiqueta < video > acepta como atributos:

  • controls:permite activar los controles del player. + info
  • poster: muestra una imagen a modo de presentación. + info
  • autoplay, loop, muted, preload y src: tienen la misma función que en la etiqueta audio.
  • width: establece el ancho del reproductor de video en píxeles. + info
  • height: establece la altura del reproductor de video en píxeles. + info

Iframe

Se utiliza para incrustar otro documento HTML que se va a cargar en forma independiente en la página actual. Podemos agregar: contenidos de terceros,
interfaces de usuario, videos de YouTube, mapas de Google Maps o banners de publicidad desde otro sitio.

< iframe src="#página_fuente.html" width="290" height="250">Texto por si el navegador no reconoce la etiqueta < /iframe>

  • width y heigth: ancho y alto.
  • frameborder: con la relación 0 el iframe es sin borde, con la relación 1 con borde.
  • scrolling: habilita las barras de desplazamiento si el contenido no cabe en el iframe.
    Con “yes” aparecen siempre, con “auto” aparecen sólo si es necesario y con “no” no aparecerán nunca.
